# 📲 Panduan Instalasi AzHua

## Persyaratan

- Android 8.0 (API 26) atau lebih tinggi
- Memori minimal 2GB RAM
- Koneksi internet untuk streaming

## Langkah Instalasi

### 1. Download APK

Kunjungi halaman [Releases](https://github.com/dhasap/azhua-app/releases) dan download file `AzHua-v1.0.0-release.apk`.

### 2. Izinkan Instalasi

Buka **Pengaturan** > **Keamanan** > **Sumber Tidak Dikenal**, lalu izinkan instalasi dari browser/file manager yang Anda gunakan.

### 3. Instal APK

Buka file APK yang sudah didownload dan tap **Instal**.

### 4. Instal Ekstensi

1. Buka aplikasi AzHua
2. Pergi ke tab **Paviliun** (ikon 📦)
3. Cari ekstensi "Anichin"
4. Tap **Download** dan tunggu proses selesai
5. Tap file APK yang diunduh untuk menginstal ekstensi
6. Kembali ke AzHua, ekstensi akan terdeteksi otomatis

### 5. Mulai Menonton

1. Pergi ke tab **Jelajah** (ikon 🔍)
2. Pilih sumber "Anichin"
3. Pilih anime yang ingin ditonton
4. Pilih episode dan nikmati!

## Troubleshooting

### "Aplikasi tidak terinstal"
- Pastikan Anda mengizinkan instalasi dari sumber tidak dikenal
- Coba hapus versi lama sebelum menginstal yang baru

### "Ekstensi tidak terdeteksi"
- Pastikan ekstensi sudah terinstal (bisa dicek di Pengaturan > Aplikasi)
- Restart aplikasi AzHua

### "Video tidak bisa diputar"
- Pastikan koneksi internet stabil
- Coba sumber/server lain jika tersedia
