# 🎬 AzHua - Aplikasi Streaming Donghua

[![Version](https://img.shields.io/badge/version-1.0.0-blue)](https://github.com/dhasap/azhua-app/releases)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

AzHua adalah aplikasi streaming Donghua (Anime China) dengan arsitektur modular yang memungkinkan penambahan sumber konten melalui sistem ekstensi.

## 🌟 Fitur Utama

- 🎨 **Tema Kultivator** - Desain gelap Obsidian & Jade yang elegan
- 🔌 **Arsitektur Modular** - Tambah sumber baru via ekstensi tanpa update app
- 🌐 **Web Scraping** - Ekstrak data dari website sumber secara real-time
- 📺 **Video Player** - ExoPlayer dengan dukungan HLS streaming
- 💾 **Riwayat Tontonan** - Database Room untuk melacak progress
- ☁️ **Paviliun Kitab** - Download ekstensi dari repository GitHub

## 🚀 Cara Instalasi

1. Download file APK dari [Releases](https://github.com/dhasap/azhua-app/releases)
2. Aktifkan "Install dari Sumber Tidak Dikenal" di Pengaturan Android
3. Buka file APK dan instal
4. Buka aplikasi AzHua
5. Kunjungi tab "Paviliun" untuk mengunduh ekstensi

## 🛠️ Teknologi

- **Kotlin** - Bahasa pemrograman utama
- **Jetpack Compose** - UI modern dan reaktif
- **Room Database** - Persistensi data lokal
- **ExoPlayer** - Pemutar video yang powerful
- **Jsoup** - Web scraping dan parsing HTML
- **PathClassLoader** - Dynamic loading ekstensi
